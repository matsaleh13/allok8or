# allok8or

## Overview

A collection of memory allocators for game development and general purpose applications.

## Status

- Still very much WIP.
- Working:
  - PagingAllocator
  - PassThroughAllocator
  - DiagnosticAllocator
  - StdAllocatorAdapter
- WIP:
  - BlockAllocator (aka pool allocator)
- Nothing Yet:
  - LineaarAllocator
  - StackAllocator (maybe)
  - FreeListAllocator (maybe)
